# Promo Asset Plan

Promo art should reuse the same visual language as the screenshots:

- dark neutral browser/player surfaces
- warm orange PiP accent from the extension icon
- clean utility typography
- no decorative gradients as the primary subject
- no real streaming-service trademarks or logos

Recommended output sizes:

| Asset         | Size       |
| ------------- | ---------- |
| Small promo   | 440 x 280  |
| Marquee promo | 1400 x 560 |

Do not overpromise unblocking. Use language like "best-effort unblocking" in
listing copy and keep promo art focused on user controls.
